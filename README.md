# PySI - Prevención y Seguridad Locativa 🛡️

**PySI** es una solución de software diseñada para la gestión, reporte y mitigación de riesgos e incidentes en infraestructuras físicas. Este proyecto se desarrolla en el marco de la asignatura **Ingeniería de Software I** y busca optimizar la seguridad locativa mediante un control riguroso de sedes, clientes y hallazgos técnicos.

## 🚀 Descripción General
El sistema permite a los usuarios (Gerentes y Secretarias) registrar incidentes de seguridad, asociarlos a sedes específicas de clientes y realizar un seguimiento hasta su resolución. El objetivo primordial es minimizar la materialización de riesgos mediante la notificación oportuna y la evidencia fotográfica.

## ✨ Funcionalidades Principales
- **Gestión de Clientes:** Registro, consulta y actualización de entidades legales.
- **Control de Sedes:** Administración de múltiples ubicaciones por cada cliente.
- **Reporte de Incidentes (RF-001):** Captura detallada de riesgos con niveles de prioridad (Bajo, Medio, Alto).
- **Evidencia Digital:** Capacidad para adjuntar soportes visuales a cada hallazgo.
- **Seguridad y Acceso:** Control de sesiones basado en roles (RBAC).
